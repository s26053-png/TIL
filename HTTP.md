# HTML 한눈에 정리

## 1. HTML이란?
- 웹 페이지의 **뼈대(구조)**를 만드는 언어
- 텍스트, 이미지, 버튼 등을 배치함.

---

## 2. 기본 구조

```html
<!DOCTYPE html>
<html>
  <head>
    <title>페이지 제목</title>
  </head>

  <body>
    화면에 보이는 내용
  </body>
</ht상l>
```

## 구상 설명
- !doctype html: HTML5 문서 선언
- html: 전체 문서
- head: 설정 정보 (제목, CSS, 메타데이터)
- body 실제 화면에 보이는 내용
---
## 3. 주요 태그
## 구조/레이아웃
- div: 구역 나누기 (블록 요소)
- span: 글자 일부 묶기(인라인 요소)
## 텍스트
- p: 문단
- a: 링크
---
## 미디어
- img: 이미지
---
## 폼 (입력)
- form: 입력 폼
- input: 입력칸
- button: 버튼
---
## 4. 시맨틱 태그 (의미 있는 구조)
- header: 상단 (헤더)
- main: 핵심 내용
- section: 구역 나누기
- footer: 하단
``` html
<header>상단</header>
<main>내용</main>
<footer>하단</footer>
```
---
## 5. 핵심 정리
- HTML: 웹 페이지 구조
- div/span: 기본 구역
- form/input: 사용자 입력
- semantic 태그: 의지 있는 구조
---
## 6. 한 줄 요약
HTML은 "웹 페이지의 뼈대를 만드는 언어"이다
