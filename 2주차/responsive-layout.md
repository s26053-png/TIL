# 반응형&레이아웃

## 1. 반응형
반응형 웹은 화면 크기에 따라 레이아웃이나 스타일이 바뀌는 웹 디자인 방식이다.

---

## 2. media Query
`media`를 사용하면 특정 화면 크기에서만 CSS를 적용할 수 있다.

## 주요 속성
- `max-width` 지정한 크기 이하에서 적용
- `min-width` 지정한 크기 이상에서 적용

---

## 3. 반응형 단위

- `%`
부모 요소 기준 비율

- `vw`
브라우저 화면 너비 기준

- `vh`
브라우저 화면 높이 기준

- `rem`
루트(html) 글자 크기 기준
---
## 4. Flex

flex는 1차원 레이아웃 정렬 방식이다.
---
## 주요 속성
- justify-content 가로 방향 정렬
- align-items 세로 방향 정렬
---# 6. Flex vs Grid

## 5. Grid
Gird는 2차원 레이아웃 시스템이다.
---
# 6. Flex vs Grid

| Flex | Grid |
| ----- | ----- |
| 1차원 | 2차원 |
| 한 방향 정렬 | 행과 열 동시 제어 |
| 간단한 정렬에 적합 | 복잡한 레이아웃에 적합 |
